# 🛒 OnlineShopList

**OnlineShopList** est une application web simple et efficace pour gérer vos listes de courses directement depuis votre navigateur.  
Elle est optimisée pour **mobile**, mais fonctionne également sur ordinateur.  

🌐 **Site en ligne** : [momort02.github.io/Onlineshoplist](https://momort02.github.io/Onlineshoplist)

---

## ✨ Fonctionnalités

- ✅ **Cases à cocher** pour marquer les articles achetés  
- 📂 **Organisation par catégories** (Fruits, Légumes, Produits laitiers, etc.)  
- 📤 **Exportation de la liste** pour la sauvegarder ou l’envoyer  
- 📱 **Envoi par WhatsApp ou Messages** en un clic  
- 🧮 **Calcul automatique du total d’articles**  
- 💾 **Sauvegarde locale** : vos données restent enregistrées même après fermeture du navigateur  

---

## 🚀 Utilisation

1. Ouvrir [le site](https://momort02.github.io/Onlineshoplist) depuis un ordinateur ou un smartphone  
2. Ajouter vos articles dans les catégories correspondantes  
3. Utiliser les cases à cocher pour suivre vos achats  
4. Exporter ou partager la liste par WhatsApp / Messages si besoin
